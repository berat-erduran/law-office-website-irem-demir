# Law Office Website – İrem Demir

## Project Overview

This project is a professional law office website developed for Attorney İrem Demir. The website was designed to provide visitors with information about legal services, practice areas, professional background and contact details through a modern and responsive interface.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap

## Features

* Responsive web design
* Modern and professional user interface
* Legal service presentation pages
* Contact and consultation section
* Mobile-friendly structure
* Optimized navigation experience

## Purpose

The purpose of this project is to establish a strong online presence for a law office and enable clients to access legal information and contact services easily.
